文字列あてゲーム！

説明

文字列あてゲームはヒントをもとに4文字のランダム半角英小文字を推測するゲームです！
文字列に同じ文字は出てきません。 解答権は20回です。
以下のようなヒントをもとに正解を導きましょう！

正解が「they」で推測した文字列が「play」だった場合

文字「y」が正解と推測語で同じ位置で使われているため 1 eat 0 bite と表示されます。


正解が「they」で推測した文字列が「year」だった場合

文字「e」と「y」が正解と推測語で使われているため 0 eat 2 bite と表示されます。
